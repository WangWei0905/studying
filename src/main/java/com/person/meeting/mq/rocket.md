
rocketmq 如何处理事物
RocketMQ 是一个开源的分布式消息队列，它提供了事务消息、顺序消息、延时消息、消息轨迹等功能。
在 RocketMQ 中，事务消息是一种特殊的消息，它需要通过一个事务步骤来完成。
事务消息分为三个阶段：预提交、提交和回滚。
